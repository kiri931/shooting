Step1: Canvasを表示する（HTML）


<!doctype html>
<html lang="ja">

<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Step1: Canvasを表示する</title>
  <link rel="stylesheet" href="style.css" />
</head>

<body>
  <main class="container">
    <h1>Step1: Canvasを表示する</h1>
    <p class="note">黒い枠（ゲーム画面）が表示されればOKです。</p>

    <canvas id="gameCanvas" width="480" height="640"></canvas>
  </main>

  <script src="main.js"></script>
</body>

</html>
  1. <!DOCTYPE html> は、このファイルが であることを示す宣言です。
  2. <html lang="ja"> は、この文書が 語で書かれていることを示します。
  3. <head> タグの中には、ページの見た目には直接表示されない 情報が書かれます。
  4. <meta charset="UTF-8" /> は、文字コードを に指定することで、日本語が文字化けしないようにします。
  5. <meta name="viewport" ... /> は、スマホ等でも見やすいように画面の拡大率（表示領域）を調整する です。
  6. <title>...</title> は、ブラウザの に表示されるタイトルを設定します。
  7. <link rel="stylesheet" href="style.css" /> は、外部の ファイルを読み込み、デザインを適用します。
  8. <body> タグには、実際に画面に表示される が書かれます。
  9. <main class="container"> は、ページの主要な内容をまとめる 要素です。
  10. <canvas id="gameCanvas" width="480" height="640"></canvas> は、ゲーム画面を描くための 領域を作ります。ここでは幅が ピクセル、高さが ピクセルに設定されています。
  11. <script src="main.js"></script> は、外部の ファイルを読み込み、ゲームの動きをプログラムします。
