🟦 段落（p）

行間・文字の見やすさが style.css で調整されています。教科書向きの読みやすいスタイルです。

これは通常の段落です。読みやすい行間と文字間が設定されています。

🟦 インラインコード（span.code-sample）

🟦 リスト（ol / ul / li）

適度な余白がついて見やすいリストになります。

順序ありリスト（ol）

  1. HTML を書く
  2. CSS を当てる
  3. JavaScript を加える

順序なしリスト（ul）

  • ブラウザ
  • テキストエディタ
  • ローカルサーバー

  • API：機能を提供する仕組み
  • プロトコル：通信のルール
