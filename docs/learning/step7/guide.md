# Step7: 弾を複数（配列）

## このStepでできること
- Spaceを押すたびに弾を増やして、複数弾を撃てます。

## 前Stepからの差分
- Step6の「1発制限（`bullets.length === 0`）」を外して、毎回`bullets.push(...)`するようにします。
- 弾の更新はすでに配列ループになっているので、そのまま複数弾で動きます。

## Playgroundで確認する（変更が見えるポイント）
- 右側のJSでSpaceを押しっぱなしにすると弾が増えすぎるので、クールダウンを入れると改善する（次のStep8/9の考え方）。
- 弾を消す条件（画面外判定）を変えると残り方が変わる。
