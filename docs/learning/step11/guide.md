# Step11: スコア表示

## このStepでできること
- 敵を倒した回数に応じてスコアが増え、画面に表示されます。

## 前Stepからの差分
- `score`変数と、加点量（例: `SCORE_PER_KILL = 100`）を追加します。
- 弾×敵の当たり判定で敵を消したタイミングで、`score += SCORE_PER_KILL`します。
- `drawHud()`（HUD描画）を追加し、左上に`SCORE: ...`を表示します。

## Playgroundで確認する（変更が見えるポイント）
- 右側のJSで`SCORE_PER_KILL`を変えると、1体あたりの加点が変わる。
- `drawHud()`の表示位置（x,y）を変えると、表示場所が動く。
