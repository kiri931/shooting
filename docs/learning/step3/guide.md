# Step3: 自機を左右に動かす

## このStepでできること
- キー入力（←→）で自機を左右に動かせます。

## 前Stepからの差分
- `keydown`イベントを追加して、左右キーで`player.x`を増減します。
- 画面外に出ないように、`player.x`を0〜`canvas.width - player.width`の範囲に収めます。
- 動いたら`draw()`を呼び直して、表示を更新します。

## Playgroundで確認する（変更が見えるポイント）
- 右側のJSで`player.speed`を変えると、移動スピードが変わる。
- 範囲チェック（画面外に出ない処理）を一時的に外すと、画面外に出る（違いが見える）。
