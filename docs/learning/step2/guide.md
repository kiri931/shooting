# Step2: 自機（四角）を表示する

## このStepでできること
- Canvas上に「自機（プレイヤー）」を四角で描きます。

## 前Stepからの差分
- **プレイヤーの状態（座標・サイズ）**を表す`player`オブジェクトを追加します。
- **描画をまとめる`draw()`関数**を追加し、背景→自機の順に描くようにします。
  - これで次のStep（移動）でも「座標を変えたら`draw()`」という形で増やしやすくなります。

## Playgroundで確認する（変更が見えるポイント）
- 右側のJSで`player.x`や`player.y`の初期値を変えて、四角の位置が変わることを確認する。
- `player.width`/`player.height`を変えて、自機の大きさが変わることを確認する。
