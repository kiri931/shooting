# Step8: 敵（直進）

## このStepでできること
- 敵が上から降ってきます（直進タイプ）。
- 自機移動＋弾もそのまま使えます。

## 前Stepからの差分
- `enemies`配列を追加し、一定確率で敵を`spawnEnemy()`で追加します。
- `updateEnemies()`で敵を下に動かし、画面外に出たら削除します。
- `draw()`で敵を描画します（画像があれば画像、なければ四角）。
- Spaceの押しっぱなしで増えすぎないよう、軽いクールダウン（連射制限）も入っています。

## Playgroundで確認する（変更が見えるポイント）
- 右側のJSで`Math.random() < 0.03`の確率を変えると、敵の出現頻度が変わる。
- `ENEMY_SPEED`を変えると、敵の落下速度が変わる。
