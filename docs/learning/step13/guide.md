# Step13: 仕上げ（完成版）

## このStepでできること
- 敵が2種類になります（直進 / 追尾）。
- Uキーで武器を強化して、弾が増えます（Lv1〜3）。

## 前Stepからの差分
- 敵タイプ`ENEMY_TYPE`を追加し、`spawnEnemy()`で直進/追尾を混ぜて出します。
- 追尾敵はプレイヤー中心へ向かうように移動計算を入れます。
- 武器レベル`weaponLevel`を追加し、Uキーで強化できるようにします。
- 発射処理を「武器レベルに応じて弾を複数生成」する形にします。
- HUDに`WEAPON: Lv...`表示を追加します。

## Playgroundで確認する（変更が見えるポイント）
- 右側のJSで`MAX_WEAPON_LEVEL`や弾の横速度（`BULLET_SPEED_X`）を変えると、散らばり方が変わる。
- 追尾速度（`ENEMY_HOMING_SPEED`）を変えると、追尾の強さが変わる。
- `Math.random() < 0.03`を変えると、敵の出現頻度が変わる。
